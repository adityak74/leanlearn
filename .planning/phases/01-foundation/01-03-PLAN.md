<task type="auto">
  <name>Task 1: Setup Auth Client and Hono Session Middleware</name>
  <files>app/lib/auth-client.ts, app/server.ts</files>
  <action>
    Configure the Better Auth client for the frontend and implement Hono middleware to set the user in the context.
    - Create `app/lib/auth-client.ts` using `createAuthClient`.
    - Implement a Hono middleware in `app/server.ts` that:
      - Calls `auth.api.getSession({ headers: c.req.raw.headers })`.
      - Sets `c.set('user', session?.user)` and `c.set('session', session?.session)`.
      - Provides these values to the React Router context (via `hono-react-router-adapter`).
  </action>
  <verify>
    <automated>grep -q "createAuthClient" app/lib/auth-client.ts && grep -q "c.set('user'" app/server.ts</automated>
  </verify>
  <done>Session info is available to both Hono and React Router loaders.</done>
</task>

  <action>
    Create the login page with Google OAuth button and the protected dashboard route.
    - Create `app/routes/login.tsx` with a button calling `authClient.signIn.social({ provider: "google" })`.
    - Create `app/routes/dashboard.tsx` that:

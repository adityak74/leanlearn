  <action>
    - Update `app/server.ts` to add `POST /api/activity/:id/complete`.
    - Secure the route with the session middleware.
    - Logic for marking completion and calculating progress:
      1. Insert into `activity_completion`.
      2. Fetch all `required` activities for the parent course.
      3. Fetch all completed `required` activities for this user/course.
      4. `progress = (completed count / total required count) * 100`.
      5. Update or insert into `course_progress` table.
      6. If progress is 100%, set `completed_at` in `course_progress`.
    - Ensure `GET /api/me/courses` and `GET /api/course/:slug` also return the current user's progress.
  </action>
